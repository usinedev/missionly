
User :
role : admin, freelance, society, accountant
langage : FR, ENG, JPN, NL, ...
priceUnity : euro, dollar, yen, ...

Mission : 
adress : (adresse) || distanciel
price : [(priceNumber), (priceUnity)]
status : created, published, started, finished
tags : [tag1, tag2, ...]



API : 
User : 
login : email, password
register : email, password, userName, role, TVAnumber

La route Delete est protégée pour que seul un Admin puisse effacer de la db une Mission
La route pour soft-delete est un post qui archive la mission (elle n'est plus lié à l'utilisateur et est mise en non publiée)

Mission : 
Get : 
user/id : id (de l'utilisateur)
/id : id (de la mission)
/name : name (de la mission)

Post : 
tags : tags (de la mission dans le body)
create : name, start, adress, price, tags, user (id)
/id/update : id (de la mission), update (dans le body)
